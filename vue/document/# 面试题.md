# 面试题

1. v-show和v-if的区别
  + v-show通过CSS display控制显示和隐藏
  + v-if罪案真正的渲染和销毁，而不是显示和隐藏
  + 频繁切换显示状态用v-show，否则用v-if
2. 为何在v-for中用key
  + 必须用key，且不能是index和random
  + diff算法中通过tag和key来判断，是否是sameNode
  + 减少渲染次数，提高渲染性能
